## 1. 📦 What is a Linux Distribution?

### 🇮🇩 Bahasa Indonesia
**Linux Distribution (Distro)** adalah sistem operasi yang dibangun di atas **Linux kernel**. Bayangkan distro seperti **cabang atau franchise dari perusahaan yang sama** — semuanya berbagi komponen inti yang sama (kernel), struktur arsitektur yang sama, dan filosofi yang sama, namun masing-masing menawarkan paket software, konfigurasi, dan pengalaman yang berbeda untuk memenuhi kebutuhan yang beragam.

Linux digunakan untuk:
- **Server** — stabil, aman, dan andal
- **Desktop** — gratis, open-source, dan sangat customizable
- **Mobile devices** — Android berbasis Linux kernel
- **Embedded systems** — router, smart TV, IoT devices
- **Cloud computing** — mayoritas server cloud berjalan di Linux
- **Cybersecurity** — open-source = bisa diaudit dan dikustomisasi sepenuhnya

### 🇬🇧 English
A **Linux Distribution (Distro)** is an operating system built on top of the **Linux kernel**. Think of distros like **branches or franchises of the same company** — they all share the same core components (kernel), organizational structure, and philosophy, but each offers its own unique software packages, configurations, and experiences tailored to diverse needs.

Linux is used for:
- **Servers** — stable, secure, and reliable
- **Desktop** — free, open-source, and highly customizable
- **Mobile devices** — Android is based on the Linux kernel
- **Embedded systems** — routers, smart TVs, IoT devices
- **Cloud computing** — majority of cloud servers run on Linux
- **Cybersecurity** — open-source = fully auditable and customizable

### 🇯🇵 日本語
**Linux ディストリビューション（ディストロ）** は**Linuxカーネル**の上に構築されたオペレーティングシステムです。ディストロは**同じ会社の支店やフランチャイズ**のようなものと考えてください — すべて同じコアコンポーネント（カーネル）、アーキテクチャ、哲学を共有しますが、それぞれ多様なニーズに応えるために独自のソフトウェアパッケージと設定を提供します。

Linux の用途：
- **サーバー** — 安定、安全、信頼性が高い
- **デスクトップ** — 無料、オープンソース、高度にカスタマイズ可能
- **モバイルデバイス** — Android は Linux カーネルベース
- **組み込みシステム** — ルーター、スマートTV、IoT デバイス
- **クラウドコンピューティング** — クラウドサーバーの大半は Linux で稼働
- **サイバーセキュリティ** — オープンソース = 完全に監査・カスタマイズ可能

## 2. 🌍 Popular Linux Distributions

### 🇮🇩 Bahasa Indonesia
Perbedaan utama antara berbagai distro Linux adalah **paket yang disertakan, antarmuka pengguna, dan tools yang tersedia**. Setiap distro dioptimalkan untuk use case yang berbeda.

### 🇬🇧 English
The main differences between distributions are the **included packages, user interface, and available tools**. Each distro is optimized for different use cases.

### 🇯🇵 日本語
各ディストリビューションの主な違いは**含まれるパッケージ、ユーザーインターフェース、利用可能なツール**です。各ディストロは異なる用途に最適化されています。

## 3. 🔐 Cybersecurity-Focused Distributions

### 🇮🇩 Bahasa Indonesia
Untuk spesialis keamanan siber, ada distro yang secara khusus dirancang dengan **tools offensive dan defensive security** yang sudah ter-install. Ini menghemat waktu setup dan memastikan semua tool yang dibutuhkan sudah tersedia.

### 🇬🇧 English
For cybersecurity specialists, there are distros specifically designed with **offensive and defensive security tools** pre-installed. This saves setup time and ensures all necessary tools are readily available.

### 🇯🇵 日本語
サイバーセキュリティの専門家向けに、**攻撃的・防御的セキュリティツール**があらかじめインストールされた特化したディストロがあります。セットアップ時間を節約し、必要なすべてのツールをすぐに利用できます。

## 4. 🔵 Debian — Deep Dive

### 🇮🇩 Bahasa Indonesia
**Debian** adalah salah satu distro Linux yang paling banyak digunakan dan dihormati. Dikenal karena **stabilitas dan keandalannya**, Debian menjadi fondasi bagi banyak distro populer lainnya seperti Ubuntu, Kali Linux, dan Parrot OS.

**Keunggulan Debian:**
- **Stabilitas tinggi** — cocok untuk server yang harus berjalan 24/7
- **Long-term support (LTS)** — update dan security patch hingga 5 tahun
- **Sangat fleksibel** — bisa dikonfigurasi untuk hampir semua use case
- **Komitmen keamanan** — track record keamanan yang solid
- **Package manager `apt`** — sistem manajemen paket yang powerful

**Kekurangan:** Kurva belajar lebih curam dibanding Ubuntu, setup dan konfigurasi bisa kompleks — namun justru memberikan kontrol penuh atas sistem.

### 🇬🇧 English
**Debian** is one of the most widely used and respected Linux distributions. Known for its **stability and reliability**, Debian serves as the foundation for many other popular distros like Ubuntu, Kali Linux, and Parrot OS.

**Debian's strengths:**
- **High stability** — ideal for servers that must run 24/7
- **Long-term support (LTS)** — updates and security patches for up to 5 years
- **Highly flexible** — configurable for almost any use case
- **Security commitment** — solid security track record
- **`apt` package manager** — powerful package management system

**Drawback:** Steeper learning curve than Ubuntu; setup and configuration can be complex — but this grants full control over the system.

### 🇯🇵 日本語
**Debian** は最も広く使われ、尊敬されているLinuxディストリビューションの一つです。**安定性と信頼性**で知られ、Ubuntu、Kali Linux、Parrot OS など多くの人気ディストロの基盤となっています。

**Debian の強み：**
- **高い安定性** — 24/7稼働が必要なサーバーに最適
- **長期サポート（LTS）** — 最大5年間のアップデートとセキュリティパッチ
- **高い柔軟性** — ほぼあらゆるユースケースに設定可能
- **セキュリティへのコミットメント** — 確かなセキュリティ実績
- **`apt` パッケージマネージャー** — 強力なパッケージ管理システム

**欠点：** Ubuntu より学習曲線が急で、設定が複雑になることがある — しかしシステムの完全なコントロールを提供します。

### APT Package Manager — Quick Reference

```bash
# Update package list
$ sudo apt update

# Upgrade all installed packages
$ sudo apt upgrade

# Install a package
$ sudo apt install <package-name>

# Remove a package
$ sudo apt remove <package-name>

# Remove + delete config files
$ sudo apt purge <package-name>

# Search for a package
$ apt search <keyword>

# Show package info
$ apt show <package-name>

# List installed packages
$ apt list --installed
```

## 🧠 Quick Reference Cheatsheet

```bash
# Check current distro info
$ cat /etc/os-release
$ lsb_release -a
$ uname -a

# Check kernel version
$ uname -r

# APT package management (Debian/Ubuntu/Kali/Parrot)
$ sudo apt update && sudo apt upgrade    # update system
$ sudo apt install <package>             # install package
$ sudo apt remove <package>              # remove package
$ apt search <keyword>                   # search packages

# Check installed packages
$ apt list --installed
$ dpkg -l

# System info
$ hostnamectl                            # OS, hostname, kernel info
$ lscpu                                  # CPU info
$ free -h                                # RAM info
$ df -h                                  # disk space
```
